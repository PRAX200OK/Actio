# actio version

Print the version number of Actio.

## Synopsis

```bash
actio version [-s|--short]
```

- **--short**, **-s** — Print only the version string (e.g. `0.1.0`) for scripting.

## Output

```
Actio v0.1.0
```

With `--short`:

```
0.1.0
```

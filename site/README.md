# Actio Framework – Documentation site

Developer documentation for the Actio (AI Context Toolkit) framework. Built with [Docusaurus](https://docusaurus.io).

## Prerequisites

- Node.js 18+

## Install and run

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm run serve
```

Static output is in `build/`.
